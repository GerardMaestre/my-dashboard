# 07 Herramientas Pro

Suite visual de herramientas avanzadas integrada en Horus Engine.

## Scripts incluidos

- Analizador_Espacio.py
  - DESC: Analizador de espacio en C:\ para detectar carpetas pesadas.
  - Modo recomendado: Integrado (terminal del dashboard).
  - Resultado: Ranking visual TOP de carpetas por tamano y resumen final.

- Desinstalador_Root.bat
  - DESC: Desinstalador visual global usando registro de Windows.
  - Modo recomendado: Visual externo.
  - Resultado: Ventana de seleccion de programas y ejecucion de desinstalador.

## Notas de uso

- Desinstalador_Root.bat puede pedir permisos UAC.
- Analizador_Espacio.py puede tardar mas en discos grandes.
- Si no aparece la herramienta, recarga el dashboard con el boton de refresco.
